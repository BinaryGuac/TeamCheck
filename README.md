# ⚽ TeamCheck — Game Day Poll

> *"Are we playing this Sunday or what?"*

We've all been there. The group chat explodes on Saturday night — *"Who's in tomorrow?"* — and somehow you still show up to find only 6 people on the pitch. Not anymore.

**TeamCheck** is a simple, no-login poll page where your crew can say whether they're showing up this Sunday (or not). Everyone sees everyone else's answer in real time. No apps to install, no accounts to create — just open the link and click.

---

## 🎯 What it does

- Shows the **next 3 upcoming Sundays** so players can plan ahead
- Each player picks their name, then says **IN** (and how many they're bringing) or **OUT**
- Everyone's answers show up live on the same page
- The captain can see at a glance if there are enough players to run a game

---

## 🕹️ How to use it

1. Open the link
2. Pick your name from the dropdown
3. For each Sunday, choose one of:
   - ⛔ **OUT** — can't make it
   - 👤 **+1 (just me)**
   - 👥 **+2, +3...** — bringing friends
4. Hit **CONFIRM** — done!

That's it. No password, no account, no app.

---

## 👀 What you'll see

- A table with all 30 roster spots and the next 3 game dates
- Everyone's current answer shown as colour-coded pills — green for IN, red for OUT
- A total headcount at the bottom of each date column so you know instantly if the game is on
- The page quietly refreshes every 5 minutes to stay current

---

## 🏗️ Under the hood (the short version)

It's a single web page backed by a Google Sheet. No servers, no databases, no fuss. The sheet stores everyone's answers; the page reads and writes to it. That's the whole thing.

---

## 👥 The Roster

30 slots for the Sunday crew — 24 regulars and 6 open spots for when new faces join the group.
